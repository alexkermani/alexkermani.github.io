Progressive Web Applications

Progressive Web Apps (PWAs) are reliable, fast, and engaging, which gives users a great experience. Here are some key characteristics of PWAs

  • Browser-installable means the app can be loaded slower once then run very quickly every following time.
  • Uses caching and offline storage to reduce load times and make the app feel native
  • Offers browser notifications that can make the user respond and reply to updates.
  • Requires a Web App Manifest (manifest.json) that defines the launch screen and how to user can navigate the app.
